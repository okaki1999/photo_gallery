# 仕様技術のバージョン
Ruby 3.1.2

rails 7.0.8

sqlite3 1.4

puma 5.0

bcrypt 3.1.7


# clone方法

GitHubからアプリをcloneします。
``` git clone https://github.com/okaki1999/photo_gallery.git ```

アプリのフォルダーに移動して下さい。
``` cd photo_gallery ```

bundle installを行ってください。
``` bundle install ```

rails db:migrateを行ってください。
``` rails db:migrate```

master.keyが必要な場合は別途ご連絡いただけると幸いです。
